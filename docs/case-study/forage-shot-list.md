# Forage — Capture Shot List

A one-pass checklist for recording every asset the case study needs.
🎥 = short video/GIF (5–12s, silent loop unless noted) · 📷 = high-res still.

## Setup (do this once before recording)
- Open the app with the **`?demo`** library so it's always full and polished.
- Use a clean, consistent **window size** (e.g. 1440×900) and the same theme per shot.
- **Hide desktop clutter**, quit notifications, and hide the cursor in stills.
- Record at 2× / Retina; export video as **MP4** (H.264), stills as **PNG**.
- Plan to **device-frame** later: browser frame, Mac window, iPhone.
- Grab **both light and dark** for the hero + a couple of key screens.

## 1 · Hero
- [ ] 🎥 Hero loop — scroll the masonry grid, hover a tile, open one item, close. *(or use a clean still)*
- [ ] 📷 Populated grid, full-bleed, the "wow" shot.
- [ ] 📷 Same view in a **browser frame** and a **Mac window frame**, side by side.

## 2 · Capture
- [ ] 🎥 Paste a URL → it **unfurls into a rich bookmark** (title + thumbnail). *(magic moment)*
- [ ] 🎥 **Drag-and-drop** images onto the window.
- [ ] 📷 Capture dialog showing detected type + collection picker.
- [ ] 📷 *(optional)* Browser extension saving a page / mobile share sheet.

## 3 · Library & grid
- [ ] 📷 The masonry grid (signature visual).
- [ ] 🎥 Density slider resizing tiles.
- [ ] 🎥 Filter bar — switch type / source / tag.

## 4 · Item detail
- [ ] 🎥 Open a tile → detail view → **eyedropper sampling a color** into the palette.
- [ ] 📷 Detail panel: palette, tags, collections, "derived from".

## 5 · Collections & color
- [ ] 🎥 **Browse by color** → pick a hue → library filters to it.
- [ ] 📷 Collections grid + the **Unfiltered** inbox tab (saves not yet filed).
- [ ] 📷 A collection cover showing mixed previews (images, font glyphs, palettes).
- [ ] 📷 A single collection page (e.g. "Studio Rebrand").

## 6 · Moodboards (centerpiece)
- [ ] 🎥 Arrange items on the canvas, add a sticky note, **draw an arrow**, **pinch-zoom**, hit **Present**.
- [ ] 🎥 **Multi-select** saves in the picker to drop a batch on at once; or **auto-moodboard** a whole collection in one click.
- [ ] 📷 A finished, beautiful moodboard.

## 7 · Storyboards
- [ ] 📷 / 🎥 A storyboard's frame sequence.

## 8 · Palettes · Fonts · Kits (design-tool features)
- [ ] 🎥 **Extract a palette from an image**, hover swatches, **click-to-copy a hex** (show the toast).

- [ ] 🎥 A **Style Kit** (palette + fonts + references) → **Export CSS**.

Caption each asset with the **why**, not the **what**.
> ✅ "Color search turns a messy library into a mood in one tap."
> ❌ "The color filter screen."
